# Device Management UI

Frontend application for the Device Management System, built with Angular.

## Prerequisites

- Node.js
- Angular CLI (`npm install -g @angular/cli`)
- Backend API running on `http://localhost:5219`

## How to run

```bash
npm install
ng serve
```

Open `http://localhost:4200` in your browser.

## Features

- View all devices in a grid layout
- Add, edit and delete devices
- View device details
- See which user is assigned to each device
